# Google Apps for OS X

A collection of lightweight, native macOS desktop apps for Google services.

Each application is packaged as a standalone macOS app and distributed as a `.dmg`, designed for fast installation and a clean, native experience.

## ✨ Features

- Native macOS-style experience for Google services  

- Standalone apps with separate builds  

- Distributed as ready-to-install `.dmg` files  

- Lightweight and fast  

- System notifications support  

- Minimal, distraction-free UI  

## 📦 Installation

1. Download the latest `.dmg` from Releases  

2. Open the file  

3. Drag the app into `Applications`  

4. Launch normally from Spotlight or Finder
